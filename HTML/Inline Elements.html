The details element:

Detail Tag

This tag contains detailed description of any topic. This tag specifies additional details that the user can open and close on demand.

Embed Tag:

The tag defines a container for an external resource, such as a web page, a picture, a media player, or a plug-in application.

Types can be text/(type of text like html), and video/mp4
